Stage 0: Getting Started with HTML

Introduction to the Web

World Wide Web

The web is a collection of html documents that are all linked together by Hyper Links.

HTML

HTML = Hyper Text Markup

Everything online is made up of HTML.

HTML is made up of:

  • Text content (what you see)
  • Markup (what it looks like)
  • References to other documents (images, videos)
  • Links

HTML Mark-up

HTML consists of opening tags and closing tags, which make up an element.

HTML tags:

  • "em"= italics
  • "b"= bold
  • "p" = paragraph
  • "h1" to "h6" = Defines HTML headings
  • "div" = defines a section in a document
  • "!DOCTYPE" = Defines the document type
  • "br" = Defines a single line break
  • "img" = Defines an image
  • "a"= anchor (this is a link)

HTML Attributes

HTML attributes are listed after the elements, but before the corresponding value. Links (a element) and images (img element) are self-closing and are considered Void tags beacuse they do not contain any content.

HTML Whitespaces

Turns everything into one line. If we want to seperate the contents into multiple lines, we write "br" which is a Void tag for "break". The "p" tag is also used and is not a Void tag because it includes content.

Inline and Block Elements

These two types of elements are used to separate content. An example of an inline element is "br" because it adds a break in the content and places the text on the next line below. An example of a block element is "p" because it creates an invisible blox around the content and also has a height and a width. Span is also an inline element and Div is a block element.

Page Structure

Websites are created like a house. The HTML is the structure, the CSS is the styling or decor, and Javascript is the functional parts or garage door.
